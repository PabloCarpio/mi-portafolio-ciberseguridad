NetAutomation Framework: Gestión de Redes mediante Python

Automatización de despliegues OSPF, gestión de VLANs y auditoría de seguridad mediante scripts personalizados.

Conexión de Script Python con Routers Cisco usando Netmiko/Napalm vía SSH, gestionando datos en formato JSON/YAML.

Capacidades del Sistema

Despliegue Masivo

Configuración de múltiples switches en segundos sin error humano, garantizando consistencia en toda la infraestructura.

Auditoría de Cumplimiento

Scripts que revisan proactivamente si todos los routers cumplen con las políticas de seguridad y las ACLs activas.

Reportes Automáticos

Generación automática de archivos Excel o JSON detallando el estado de salud, rendimiento y configuraciones de la red.

El Desafío Técnico

Problema: Configuración Manual Laberíntica

La configuración manual de redes complejas es tediosa, propensa a errores y extremadamente lenta. Cada cambio requiere acceso individual a cada dispositivo, aumentando el riesgo de inconsistencias y fallos de seguridad.

Solución: Automatización Inteligente

Mediante el uso de librerías de Python especializadas en redes como Netmiko y Napalm, se desarrolló un framework capaz de interactuar con dispositivos de red vía SSH. Esto redujo el tiempo de despliegue y configuración en un 90%, permitiendo enfocarse en tareas estratégicas en lugar de operativas repetitivas.

user@netautomation: ~
~# python deploy_ospf.py --routers all --area 0
[INFO] Conectando a Router_Carpio_01...
[SUCCESS] Router_Carpio_01: OSPF Neighbor established.
[INFO] Conectando a Core_Switch...
[WARNING] Port 80 open on Core_Switch.
[INFO] Conectando a Router_Central_02...
[SUCCESS] Router_Central_02: OSPF Configuration applied.
~# # Proceso completado exitosamente.
